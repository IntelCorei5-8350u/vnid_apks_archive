# vnid_apks_archive

> [!NOTE]
> Nguồn chính thức để tải VNeID trên Android: [CH Play](https://play.google.com/store/apps/details?id=com.vnid)

### Cách 1: (Khuyên dùng, đây là cách mình lấy được các split APK này)
